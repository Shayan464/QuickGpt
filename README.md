## 🛠️ Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

---

## 📁 Project Structure

client/ → Frontend
server/ → Backend

---

## ⚙️ Installation

### 1. Clone the repository

git clone https://github.com/your-username/QuickGPT.git

### 2. Install dependencies

cd client
npm install

cd ../server
npm install

### 3. Run the app

# frontend

cd client
npm run dev

# backend

cd server
npm run dev

---

## 🔐 Environment Variables

Create a `.env` file in the server folder and add:

MONGO_URI=your_mongodb_url
PORT=5000
